| 7 | 账号管理 | 为应用内各模块涉及账号管理及账号信息提供技术支持 |  |

| 10 | 地图&LBS | 为应用内地图及位置信息服务提供技术支持 |  |
| 11 | 版本管理 | 提供应用内所有模块的版本信息，为应用内涉及版本信息相关的功能提供技术支持 |  |
| 12 | 消息(静默)推送 | 为应用提供消息推送，包括外部 Push 消息、内部 Push 消息及静默 Push 消息 |  |
| 13 | AR/VR | 为应用提供 AR/VR 等技术支持 |  |

| 16 | 支付 | 为应用内支付相关业务提供技术支持 |  |
| 17 | 新技术(集合) | 为应用内苹果官方或三方各类新技术应用提供技术支持（如Siri语音唤起应用内功能等） |  |
