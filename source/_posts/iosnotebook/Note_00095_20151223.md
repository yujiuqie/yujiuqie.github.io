### 五、详细内容

正常的 *if let* 用法
```swift
if let type = json.type  {

} else {
    //There is no type in the root element
}
```

*if not let* 实现
```swift
func ifNotLet<T>(optional: T?, closure: @autoclosure () -> ()) -> T {
    switch optional {
    case .None:
        closure()
    case let .Some(value)
        return value
    }
}

let type = ifNotLet(json.type) {
    XCTFail("There is no type in the root element")
}
```
